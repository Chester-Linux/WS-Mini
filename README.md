# WS-Mini
Uma estação meteorológica feita com Arduino Nano, DHT22, BMP180, e o HC-05

Se acontecer o erro de "Permission denied on /dev/ttyACM0"
use o código "sudo chmod a+rw /dev/ttyACM0" no terminal
(no Linux)
